# DOM - Random Background Color Boxes

A simple JavaScript DOM exercise that assigns a random RGB background color to each of 5 boxes on page load.

## 🚀 How It Works

- 5 boxes are rendered using HTML & CSS Flexbox
- On every page load, JavaScript selects all `.box` elements using `querySelectorAll`
- A random RGB color is generated for each box and applied via `style.backgroundColor`

## 💡 Concepts Used

- DOM Selection (`querySelectorAll`)
- `forEach` loop
- Random number generation (`Math.random`, `Math.floor`)

## 🛠️ How to Run

1. Clone or download the repository
2. Open `index.html` in any browser
3. Refresh the page to see new random colors each time!
