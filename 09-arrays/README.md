# JavaScript Arrays 🚀

This folder covers array concepts in JavaScript from basics to advanced methods.
## 📚 Concepts Covered

### 🔹 Array Loops
- for loop
- forEach()
- for...in
- for...of

### 🔹 Array Methods
- toString()
- join()
- concat()
- splice()

### 🔹 Advanced Methods
- map()
- filter()
- reduce()

## 💻 Key Learnings

- Arrays store multiple values
- Different loops help iterate data
- Built-in methods simplify operations
- map/filter/reduce are powerful tools

## ▶️ How to Run

1. Open `index.html`
2. Link one JS file at a time
3. Open console
