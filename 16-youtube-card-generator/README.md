# 🎬 YouTube Card UI (JavaScript DOM Project)

This project demonstrates how to dynamically create a YouTube-style video card using JavaScript DOM manipulation.

---

## 📘 Features

- Create video cards dynamically using JavaScript
- Display:
  - Thumbnail image
  - Video title
  - Channel name
  - Views (formatted as K/M)
  - Upload time
  - Duration overlay
- Styled using Flexbox layout
- Responsive-friendly structure

---

## 🧠 Concepts Used

### 🔹 JavaScript
- DOM Manipulation
- createElement()
- appendChild()
- setAttribute()
- Conditional logic

### 🔹 UI Logic
- View count formatting (K / M)
- Dynamic content rendering

### 🔹 CSS (Inline)
- Flexbox layout
- Positioning (absolute/relative)
- Styling components

---

## 🚀 How It Works

- A function `createCard()` is created
- It takes video details as parameters
- Dynamically generates DOM elements
- Appends structured card into container

## 🚀 Key Learnings

- Building UI using pure JavaScript
- Structuring components dynamically
- Handling real-world data formatting
- Understanding layout behavior
