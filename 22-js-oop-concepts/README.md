# 🧠 JavaScript OOP Practice (User & Animal Classes)

This project demonstrates core Object-Oriented Programming (OOP) concepts in JavaScript including:

- Classes
- Constructors
- Getters & Setters
- Inheritance
- Method Overriding

---

## 🚀 Features

### 👤 User Class
- Uses getter and setter for `name`
- Validates name length (minimum 4 characters)
- Demonstrates encapsulation using `_name`

### 🐾 Animal Class
- Base class with methods like `eats()` and `jumps()`

### 🦁 Lion Class (Inheritance)
- Extends `Animal`
- Overrides `eats()` method
- Uses `super` to call parent methods

---

## 💻 How to Run

1. Open `index.html` in your browser
2. Open Developer Console (F12)
3. View outputs and logs

---

## 🧠 Concepts Covered

- OOP in JavaScript
- Encapsulation using getters/setters
- Inheritance with `extends`
- Method overriding
- `super` keyword

---

## 📌 Learning Outcome

This project helps in understanding how real-world entities can be modeled using classes and how inheritance allows code reuse and extension.
