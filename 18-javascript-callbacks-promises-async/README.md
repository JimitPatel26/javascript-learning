# ⚡ JavaScript Callbacks, Promises & Async Project

This project demonstrates core asynchronous JavaScript concepts including callbacks, promises, and execution behavior.

---

## 📘 Features

- Custom Promise creation
- Handling resolve and reject cases
- Using `Promise.race()` for concurrent execution
- Demonstrating async behavior using `setTimeout`
- Callback-based dynamic script loading
- Understanding JavaScript execution order

---

## 🧠 Concepts Used

### 🔹 Promises
- new Promise()
- resolve() and reject()
- then() and catch()
- Promise.race()

### 🔹 Callbacks
- Passing functions as arguments
- Executing callback after async task

### 🔹 Async Behavior
- setTimeout()
- Execution order (sync vs async)

### 🔹 DOM Manipulation
- createElement()
- appendChild()

---

## 🚀 How It Works

### 1. Promises
- Two promises (`prom1`, `prom2`) are created
- Each resolves/rejects randomly
- `Promise.race()` returns the fastest result

### 2. Execution Flow
- Logs demonstrate JavaScript’s non-blocking behavior
- `setTimeout` runs after main thread execution

### 3. Callback Example
- `loadScript()` dynamically loads a script
- Callback function executes after loading

- Difference between synchronous and asynchronous code
- How promises handle async operations
- How JavaScript event loop works (basic idea)
- Using callbacks vs promises
