# JavaScript DOM Selectors

This project demonstrates selecting and styling HTML elements using JavaScript DOM methods.

## 📘 Topics Covered

### 🔹 DOM Selection
- getElementsByClassName
- getElementById
- querySelector
- querySelectorAll

### 🔹 Styling Elements
- Changing background color
- Applying styles using JavaScript

### 🔹 Iteration
- Using forEach on NodeList

## 🚀 Key Learnings

- Difference between single and multiple element selectors
- Using querySelector vs querySelectorAll
- Applying styles dynamically to multiple elements
- Iterating over DOM elements
