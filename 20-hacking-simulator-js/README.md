# 💻 JavaScript Hacking Simulator

This project simulates a terminal-style hacking interface using JavaScript, featuring typing animations and random delays to mimic real-world hacking visuals.

---

## 📘 Features

- Terminal-style UI (black background, green text)
- Typing animation effect
- Random delay between messages
- Sequential execution using async/await
- Realistic hacking simulation messages

---

## 🧠 Concepts Used

### 🔹 JavaScript
- DOM Manipulation
- createElement()
- innerHTML
- setInterval()
- setTimeout()

### 🔹 Async Programming
- Promises
- async/await
- Sequential execution of tasks

### 🔹 UI Effects
- Typing animation
- Cursor simulation

---

## 🚀 How It Works

1. Messages are displayed one by one
2. Each message appears with a typing animation
3. Random delay (1–4 seconds) is added between steps
4. Async/await ensures proper sequence
