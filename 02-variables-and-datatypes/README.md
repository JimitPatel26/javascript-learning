# JavaScript Variables & Data Types 🚀

This project covers the basics of variables and data types in JavaScript.

## 📚 Concepts Covered

- Variables: `let`, `const`, `var`
- Data Types:
  - String
  - Number
  - Boolean
  - Undefined
  - Null
- `typeof` operator
- Block scope
- Objects and object manipulation

## 💻 Key Learnings

- `let` is block scoped
- `const` cannot be reassigned
- Objects can be modified even if declared with `let`
- JavaScript is dynamically typed

## ▶️ How to Run

1. Open `index.html` in browser
2. Open Developer Tools → Console
