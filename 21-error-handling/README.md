# 🧮 Error Handling Sum App

This is a simple JavaScript project that demonstrates how to use error handling in JavaScript using `try...catch`, `throw`, and `finally`.

## 🚀 Features

- Takes two inputs using `prompt()`
- Validates user input (checks if input is a number)
- Throws custom error for invalid input
- Demonstrates runtime error clhandling using `try...catch`
- Uses `finally` block to execute code regardless of error

## 🧠 Concepts Covered

- Error handling in JavaScript
- `try...catch...finally`
- `throw new Error()`
- Input validation using `isNaN()`
- Type conversion using `Number()`

## 💻 How to Run

1. Open `index.html` in your browser
2. Enter two numbers when prompted
3. Check the console for output
