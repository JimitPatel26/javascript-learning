# Business Name Generator 🚀

This is a simple JavaScript mini project that generates random business names using basic programming concepts.

## 📚 Concepts Used

- Functions
- Variables
- if-else conditions
- Math.random()
- String concatenation

## 💡 How It Works

1. A function generates a random number (1–3)
2. Based on the number:
   - Selects an adjective
   - Selects a shop name
   - Selects a suffix
3. Combines them to form a business name

## 🧪 Example Output

- Crazy Engine Bros  
- Amazing Foods Hub  
- Fire Garments Limited  

## ▶️ How to Run

1. Open `index.html`
2. Open Developer Console
3. Refresh to generate new names
