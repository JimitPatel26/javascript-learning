# JavaScript Basics - Browser Interaction 🚀

This project demonstrates basic JavaScript concepts using browser interaction methods like alert, prompt, and confirm.

## 📚 Concepts Covered

- `alert()` → Display popup messages
- `console.log()` → Print output in browser console
- `prompt()` → Take input from user
- `confirm()` → Get user confirmation (OK / Cancel)
- Variables (`let`)
- Basic `if-else` conditions

## 💻 How It Works

1. When the page loads, an alert message is shown.
2. Messages are printed in the browser console.
3. User is asked to enter a number using prompt.
4. The entered value is displayed in console.
5. A confirmation popup appears.
6. Based on user choice:
   - OK → "You chose OK"
   - Cancel → "You chose Cancel"

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Open Developer Tools → Console to see output
