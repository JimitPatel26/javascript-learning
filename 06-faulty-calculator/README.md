# Faulty Calculator 🤯

This is a fun JavaScript mini project where a calculator sometimes gives wrong answers.

## 📂 Folder Structure

06-faulty-calculator/
│
├── index.html
└── script.js

## 📚 Features

- Takes input from user
- Performs basic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- 10% chance to give wrong results 😅

## 💡 Logic

- Uses `Math.random()`
- If random value < 0.1 → wrong calculation
- Else → correct calculation

## ▶️ How to Run

1. Open `index.html`
2. Enter two numbers
3. Check console output
