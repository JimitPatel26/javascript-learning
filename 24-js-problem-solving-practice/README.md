# 🚀 JavaScript Problem Solving Practice

This project contains multiple JavaScript problems covering core concepts like:

- Arrays & Loops
- Strings
- Objects
- Async/Await & Promises
- Local Storage
- ES6 Features

---

## 📚 Problems Included

### 🧙 Magical Sorting Hat
- Assigns students to houses based on name length

---

### 🔁 Double Trouble
- Doubles array elements
- Handles consecutive duplicates intelligently

---

### 🪞 Mirror Mirror
- Creates a mirrored string  
- Example: "abc" → "abccba"

---

### 🔐 Password Validator
- Minimum 8 characters  
- Must include:
  - Uppercase
  - Lowercase
  - Number  

---

### ➕ Sum Selector
- Adds numbers until a negative number appears

---

### 🔤 Vowel Counter
- Counts vowels in a string (case-insensitive)

---

### 💾 Local Storage Manager
- Saves notes into browser local storage

---

### ⚡ Async Array Mapping
- Multiplies array elements by 2 with delay using Promises

---

### 🛒 Async Shopper
- Simulates order placement with random delay

---

### ☕ Coffee Machine
- Simulates coffee brewing asynchronously

---

### 🔍 Product Filter
- Filters products based on price (> 5000)

---

## 💻 How to Run

1. Open `index.html`
2. Open browser console (F12)
3. Uncomment any function in `script.js`
4. Run and observe output
