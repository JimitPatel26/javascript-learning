# JavaScript Conditions & Operators 🚀

This project demonstrates conditional statements and operators in JavaScript.

## 📚 Concepts Covered

- `if`, `else if`, `else`
- Comparison operators (`==`, `===`)
- Arithmetic operators (`+`, `-`, `*`, `/`, `%`)
- Ternary operator (`? :`)

## 💻 Key Learnings

- Use `===` instead of `==` for safer comparisons
- Conditions control program flow
- Ternary operator is a short form of if-else

## ▶️ How to Run

1. Open `index.html`
2. Open Developer Console
