# JavaScript Functions 🚀

This project demonstrates how functions work in JavaScript.

## 📚 Concepts Covered

- Function declaration
- Function parameters
- Default parameters
- Return values
- Arrow functions

## 💻 Key Learnings

- Functions help reuse code
- Parameters allow dynamic input
- Default values prevent errors
- Arrow functions are modern and concise

## ▶️ How to Run

1. Open `index.html`
2. Open browser console
