# 🌐 JavaScript Async/Await & Fetch API Project

This project demonstrates how to use async/await with the Fetch API to handle asynchronous operations and retrieve data from an external API.

---

## 📘 Features

- Fetch data from an external API
- Use async/await for cleaner asynchronous code
- Handle JSON data
- Understand execution flow in async functions

---

## 🧠 Concepts Used

### 🔹 Async/Await
- async functions
- await keyword
- handling asynchronous operations

### 🔹 Fetch API
- fetch()
- working with HTTP requests
- parsing JSON data

### 🔹 Promises
- implicit promise handling with async/await

---

## 🚀 How It Works

1. `getData()` fetches data from the Pokémon API
2. The response is converted into JSON
3. `main()` waits for the data using `await`
4. Logs show execution order and async behavior

---

## 🚀 Key Learnings

- How async/await simplifies promise handling
- Working with real APIs
- Understanding non-blocking execution
- Managing asynchronous workflows
