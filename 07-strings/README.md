# JavaScript Strings 🚀

This project demonstrates how to work with strings in JavaScript.

## 📚 Concepts Covered

- String indexing
- String length
- String concatenation
- Template literals (`` ` ` ``)
- String methods:
  - `toUpperCase()`
  - `toLowerCase()`
  - `slice()`
  - `replace()`
  - `concat()`

## 💻 Key Learnings

- Strings are indexed starting from 0
- Strings are immutable (cannot be changed directly)
- Template literals are cleaner than concatenation
- Built-in methods make string manipulation easy
